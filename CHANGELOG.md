# 更新日志
本文件记录 "前端百宝箱" 扩展的所有重要变更。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范，
版本号采用 [语义化版本](https://semver.org/lang/zh-CN/) 标准。

## [2.0.0] - 2026-4-20
### 新增
- **代码片段管理增强** — 片段支持标签分类（逗号分隔输入）、模板变量、使用频率追踪（按使用次数降序排列，最常用的排在最前面）
- **中文变量翻译增强** — 新增约 80 个常用编程词组字典（用户名→username、价格→price、搜索框→searchBox 等），优先词组级最长匹配，再逐字匹配，翻译质量大幅提升；支持自定义词典和撤销黑名单参数（`userDict` / `blacklist`）
- **单元测试** — 新增 Mocha 测试覆盖核心纯函数模块：toolbox（JSON→TS、CSS 单位转换、颜色转换、时间戳、编解码、SVG 优化、console 管理）、ChineseToEnglish（单字/词组/混合翻译）、RegexVisualizer（Token 解析、匹配分析），共 98 个测试用例
- **国际化基础设施** — 新增 `src/i18n/` 模块（zh-CN / en 字符串映射）和 `t()` 翻译函数，根据 VS Code 语言环境自动切换；新增 `package.nls.json` / `package.nls.en.json` NLS 文件
- **中文代码片段触发器** — 在 js/ts/jsx/tsx/vue 文件中输入中文关键词自动展开对应代码片段（如输入 `判断` → `if (condition) { }`），覆盖 16 个分类共 150+ 个关键词：条件判断、循环、函数、类与对象、异常处理、定时器、DOM 事件、Promise & Async、JSON & 数据、TypeScript、React、Vue、导入导出、注释、调试、常用工具、数组操作、字符串处理、请求/API、工具函数、表单处理、路由、存储、正则表达式、DOM 操作、测试、HTML/Vue 模板
- 支持命令面板搜索「中文代码片段列表」查看所有可用片段

### 新增命令
- `extension.listChineseSnippets` — 中文代码片段列表

## [1.1.26] - 2026-4-15
### 新增
- **中文变量名自动转英文** — 在 js/ts/vue 文件中输入 `const/let/var` 后跟中文变量名时，自动翻译为英文 camelCase 命名（如 `const 用户名` → `const userName`）。默认使用本地 480 字字典（零延迟），可选开启 AI 翻译模式获得更精准结果
- **正则常用模板库** — 正则可视化测试器新增 11 个常用正则模板（邮箱、手机号、URL、身份证号、IP、日期、中文、HTML 标签、密码强度、十六进制颜色、车牌号），选择模板自动填充
- **Console.log 管理增强** — 新增「注释所有 console」「高亮所有 console」（3秒橙色高亮）、「切换注释状态」三个命令，移除功能扩展支持 console.warn/error
- **代码片段导入导出** — 支持将代码片段导出为 VS Code `.code-snippets` 格式文件，支持从 `.code-snippets` 文件导入（自动去重合并）
- **AI Class 命名多规范** — AI 生成 class 名称时可选命名规范：自定义语义化、BEM（block__element--modifier）、CSS Modules（camelCase）、Tailwind 工具类
- **JSON→TS 增强** — 新增 Zod Schema 生成模式（TS 类型 + Zod 校验）、支持从剪贴板自动读取 JSON、可自定义根类型名称
- **CSS 冗余检测一键清理** — 冗余报告面板新增全选/反选/批量删除功能，支持跨文件批量删除冗余 class 定义（删除前验证行内容防止误删）
- **图片分析压缩建议** — 图片分析报告新增格式转换建议（PNG→WebP 预估节省 ~30%、JPEG→WebP 预估节省 ~25%），大图额外显示 tinypng.com / squoosh.app 压缩工具提示
- **Figma 设计令牌提取** — 从 Figma 文件中提取颜色、字体、圆角、间距等设计令牌，支持输出为 CSS Variables / SCSS Variables / Tailwind Config 格式

### 优化
- Console.log 移除功能扩展支持 console.warn 和 console.error

### 配置新增
- `generateCssTree.autoTranslateVariable` — 是否开启中文变量名自动翻译（默认 true）
- `generateCssTree.autoTranslateUseAI` — 中文变量名翻译是否使用 AI（默认 false，使用本地字典）
- `generateCssTree.namingConvention` — AI 生成 class 名称的命名规范（默认 custom）

### 新增命令
- `extension.commentAllConsoleLogs` — 注释所有 console
- `extension.highlightConsoleLogs` — 高亮所有 console
- `extension.toggleConsoleLogs` — 切换 console 注释状态
- `extension.exportSnippets` — 导出代码片段
- `extension.importSnippets` — 导入代码片段
- `extension.figmaExtractTokens` — Figma 设计令牌提取

## [1.1.25] - 2026-4-14
### 优化
- **Figma API 限流优化** — 智能限流控制，最小请求间隔从 5 秒增加到 10 秒，支持服务端 Retry-After 响应头自动等待，指数退避重试策略（10s/20s/40s）
- **文件系统持久化缓存** — Figma 文件数据缓存 1 小时，节点数据缓存 30 分钟，缓存跨会话持久化，大幅减少 API 调用
- **智能数据获取策略** — 优先获取整个文件数据再提取指定节点，减少对 Figma API 的直接调用
- **429 错误提示优化** — 显示配额重置时间、当前配置值，提供明确的解决方案建议

### 配置新增
- `generateCssTree.figmaMinRequestInterval` — Figma API 最小请求间隔（毫秒），默认 10000ms（10秒）

## [1.1.23] - 2026-4-13
### 新增
- **HTML 无障碍（a11y）检查** — 自动检测 HTML/Vue 文件中的常见无障碍问题（img 缺 alt、button 空、input 缺 label、a 缺 href 等），使用 VS Code 原生波浪线提示，切换编辑器和保存时自动检查
- **JSON Path 查询器** — 打开可视化面板，输入 JSON 数据和路径表达式（如 `data.list[0].name`），实时预览匹配结果，支持复制
- **API 请求代码生成** — 选中 JSON 响应数据（或粘贴），选择 axios/fetch，AI 自动生成完整的 TypeScript 请求代码 + 类型定义
- **代码差异对比** — 选中两段代码（或从剪贴板获取第二段），AI 生成语义化的 Markdown 差异说明，适用于 PR 描述
- **图片资源分析** — 扫描工作区中的未使用图片、重复图片（基于 SHA-256）、超大图片，生成可视化分析报告
- **CSS 冗余检测** — 扫描 CSS/LESS/SCSS/Vue 文件中未被模板引用的 class 定义，支持当前文件和整个工作区两种模式，生成可视化报告
- **正则可视化测试** — 正则测试器升级，新增正则表达式 Token 彩色解析（字面量、字符集、捕获组、量词等）、测试文本捕获组高亮标注、图例说明

### 配置新增
- `generateCssTree.largeImageThreshold` — 大图检测阈值（字节），默认 500KB

## [1.1.20] - 2026-4-9
### 新增
- **JSON转TypeScript类型** — 选中JSON数据或粘贴JSON，自动生成 TypeScript interface/type 类型定义
- **CSS单位批量转换** — 选中CSS代码，一键转换 px↔rem、px↔vw，支持自定义基准值
- **颜色格式转换** — 选中颜色值，在 HEX/RGB/RGBA/HSL/HSLA 之间互转
- **时间戳/日期转换** — 选中时间戳或日期字符串，自动识别并双向转换，结果复制到剪贴板
- **编码/解码转换** — 选中文本，支持 URL编码/解码、Base64编码/解码、HTML实体编码/解码
- **SVG优化压缩** — 选中SVG代码，移除注释、metadata、冗余属性，压缩体积
- **正则表达式测试器** — 打开可视化面板，实时输入正则和测试文本，高亮匹配结果并展示捕获组
- **代码片段收集** — 选中代码右键保存为片段，通过侧边栏管理、插入、删除，数据持久化存储
- **占位图片生成** — 快速选择尺寸，插入 `<img src="https://placehold.co/..." />` 标签
- **Console.log管理** — 光标放在变量上右键插入 console.log，或一键移除当前文件所有 console.log
- 侧边栏布局重新设计：双列网格 + 紧凑列表，展示更多功能且不显拥挤

### 配置新增
- `generateCssTree.remBase` — px转rem的基准值（默认16）
- `generateCssTree.vwDesignWidth` — px转vw的设计稿宽度（默认375）
